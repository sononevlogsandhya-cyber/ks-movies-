KS MOVIES LAUNCH GUIDE

🚀 Launch Karo Aaj!

Step-by-step guide — Vercel deploy se lekar APK banane tak. Total time: 30 minutes!

1
Vercel Pe Upload Karo

📦 Ye Sab Files Upload Karo

Sirf ye files chahiye vercel mein — sab ek folder mein honi chahiye

manifest.json
sw.js
ks-tv.js
ks-pwa.js
vercel.json
index.html
user.html
login.html
player.html
onboarding.html
live.html
kspay.html
icons/ folder
🌐

Vercel.com pe jao

vercel.com kholo → Login karo (GitHub se) → "Add New → Project" click karo

📂

GitHub repo se deploy karo (Recommended)

Sab files GitHub pe daalo → Vercel se connect karo → Deploy hoga automatically. Ya seedha vercel deploy CLI se kar sakte ho.

✅

URL milegi — Test karo

Deploy ke baad https://ks-movies.vercel.app jaisi URL milegi. Browser mein open karo aur check karo sab pages work kar rahe hain.

2
PWA Test Karo (Browser Mein)
🔍

Chrome DevTools mein check karo

Chrome open karo → F12 → Lighthouse tab → "Progressive Web App" check karo → Score 90+ aana chahiye

📱

Mobile pe install karo (test ke liye)

Android phone mein Chrome se apni Vercel URL kholo → Address bar mein "Install App" icon dikhega → Install karo → Test karo

⚠️ Important: HTTPS zaruri hai PWA ke liye. Vercel automatically HTTPS deta hai — toh koi tension nahi!
3
APK Banao — PWABuilder Se (Free!)
🌍

PWABuilder.com kholo

Browser mein pwabuilder.com kholo — Microsoft ka official free tool hai

🔗

Apni URL daalo

Text box mein apni Vercel URL daalo (jaise https://ks-movies.vercel.app) → "Start" click karo

🤖

"Android" choose karo → Download

Packages screen pe Android ka button dikhega → Click karo → Options mein sab default chhod do → "Download" click karo → .zip milega

📦

ZIP se APK nikalo

Zip extract karo → Andar app-release-unsigned.apk milega → Direct install ho sakta hai! Ya Play Store ke liye sign karo.

4
Android TV Pe Install Karo
📺

APK TV pe transfer karo

APK file ko pendrive/SD card mein daalo → TV se connect karo → File Manager se install karo. Ya ADB use karo: adb install app.apk

🎮

Remote control navigation

KS TV.js automatically D-pad navigation enable kar deta hai! Up/Down/Left/Right arrows se navigate hoga, OK button se select hoga, Back button se wapas jayega.

🎬

Player controls

Video player mein: Play/Pause button → play/pause | → arrow → 10 sec forward | ← arrow → 10 sec rewind

5
Optional — Play Store Submit
🏪

Google Play Console

play.google.com/console → $25 one-time fee → APK sign karo → Upload karo → Review process 1-3 din

🔑

APK Sign karna (Play Store ke liye)

PWABuilder se signed APK bhi milta hai — unke wizard mein keystore generate karne ka option hai. Direct Play Store ready AAB bhi download kar sakte ho!

🎉 Aaj Ka Target

Vercel deploy + PWABuilder APK = 30 minutes mein launch ready!
