# 🎬 KS Movies — OTT Streaming Platform

A Netflix-style Indian OTT platform with movies, web series, live TV, cricket streaming and AI assistant.

## Files
- `index.html` — Entry point / Router
- `user.html` — Main home page (movies, AI bot)
- `login.html` — Login / Register
- `onboarding.html` — Language selection (first time)
- `player.html` — Video player
- `player-secure.html` — Secure player
- `live.html` — Live TV
- `ks-cricket.html` — Cricket streaming
- `kspay.html` — Payment page
- `ks-admin-panel.html` — Admin dashboard
- `privacy.html` — Privacy policy
- `ks-protect.js` — Page protection script

## Deploy
Hosted on Vercel — auto deploys on GitHub push.
